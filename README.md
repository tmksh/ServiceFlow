# ServiceFlow

生活サービス業（不用品回収・引っ越し・ハウスクリーニング等）に特化した、SaaS型 自動顧客管理プラットフォーム。

LINE連携による案件自動登録、カレンダー形式のスケジュール管理、リアルタイム通知、経営ダッシュボードを一元化し、現場の業務効率を抜本的に改善します。

## 技術スタック

| レイヤー | 技術 |
|:---|:---|
| フロントエンド（Web） | Next.js / React / TypeScript / Tailwind CSS |
| フロントエンド（モバイル） | React Native / Expo / TypeScript |
| バックエンド | Node.js / Express / TypeScript |
| データベース | MySQL |
